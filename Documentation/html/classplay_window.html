◆ call()

void playWindow::call ( unsigned *  tgt,
sf::RenderWindow &  window 
)

runs the event loop for the game play
-a bowler type (FAST/SPIN) is chosen after each over
-each ball is a random spin or swing depending on bowler type and has a random marker, few balls are wide
-batsman movements are updated, and a hit is counted when the ball falls in the bounds of the bat
-earlier hits are given 4/6 runs, late hits 1/2 runs, wickets are down when the ball hits it
-messages are displayed and music played after each 4/6/wicket

Parameters
windowaccepts a window reference to display the contents
tgtaccepts a pointer to the target to be chased and the number of overs
Note
Hit as early as you can after the ball hits the marker to get more runs
